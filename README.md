# 🎓 Student Management System

A full-stack **Student Management System** built using **Python Flask** and **PostgreSQL**. The application allows users to securely manage student records with authentication, analytics, and a responsive user interface.

# ✨ Features

- 🔐 User Registration & Login
- 🔒 Secure Password Hashing
- 👤 User Profile
- ➕ Add Student
- ✏️ Edit Student
- ❌ Delete Student
- 🔍 Search Students
- 📊 Dashboard Analytics
- 📈 Pie Chart & Bar Chart
- 👥 User-wise Student Records
- ☁️ PostgreSQL Cloud Database
- 📱 Fully Responsive Design
- 🌐 Deployed on Render

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/ErLalit18/Student-Management-System.git
```

Go inside the project

```bash
cd Student-Management-System
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```
